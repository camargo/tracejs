# trace.js<br>
A Browser Based Ray Tracer<br>
<br>
To install karma with npm:<br>
    npm install<br>
    npm install -g karma-cli<br>
<br>
To run unit tests type:<br>
    karma start karma.conf.js<br>
<br>
To install gulp with npm:<br>
    npm install<br>
    npm install -g gulp<br>
<br>
To run type:<br>
    gulp<br>
